### Struktur Folder

`main.java` --> kode utama untuk run projek

`/system` --> tempat kode untuk sistem game. contoh : `createWindow`

`/game` --> berisi kode yang berhubungan langsung dengan game. Contoh : `playerSaveData`

`/asset` --> berisi media seperti gambar, audio atau pun font.
